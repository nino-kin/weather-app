## Add Flutter packages

```bash
flutter pub add http
flutter pub add geolocator
flutter pub add geocoding
flutter pub add lottie
```

You can find the following packages in the pubspec.yaml.

```yaml
dependencies:
  flutter:
    sdk: flutter

  cupertino_icons: ^1.0.2
  http: ^1.1.0
  geolocator: ^10.1.0
  geocoding: ^2.1.1
  lottie: ^2.6.0
```
